网站项目说明

- 本项目是一个网站（website）。
- 使用 Git 进行版本控制，并托管到 GitHub。

提交到 GitHub（命令示例）
1) 初始化仓库（已初始化可跳过）:
   git init

2) 添加并提交:
   git add .
   git commit -m "docs: 更新 readme，标注为网站并准备推送 GitHub"

3) 关联远程（将 YOUR_GITHUB_REPO_URL 替换为你的仓库地址）:
   git remote add origin YOUR_GITHUB_REPO_URL

4) 推送:
   git branch -M main
   git push -u origin main
